# global-order-performance-dashboard
Interactive dashboard for analyzing global order performance using Google Looker Studio.

## Project Overview

This project was developed as part of the MySkill Data Visualization Mini Course using Google Looker Studio. The dashboard analyzes global order performance by visualizing key business metrics and trends from a transactional dataset.

## Tools Used

- Google Looker Studio
- Microsoft Excel

## Key Metrics

- Total Orders: 51,294
- Total Customers: 51,290
- Total Countries: 147
- Total Regions: 15

## Dashboard Features

- KPI Cards for business performance monitoring
- Orders by Region analysis
- Top 10 Countries by Orders
- Customer Segment distribution
- Order Priority analysis
- Orders by Ship Mode analysis
